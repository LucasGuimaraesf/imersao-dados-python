# imersao-dados-python

Dashboard de Análise de Salários na Área de Dados

Resumo:
Dashboard interativo desenvolvido em Python para análise de salários na área de dados, utilizando Streamlit, Pandas e Plotly. A aplicação permite explorar dados salariais por meio de filtros dinâmicos e visualizações interativas, facilitando a análise de tendências e características do mercado de trabalho em dados.

Este projeto foi desenvolvido como parte da Imersão de Python com Dados da Alura, com o objetivo de aplicar conceitos de análise de dados e desenvolvimento de dashboards interativos utilizando Python.

O dashboard permite explorar informações salariais da área de dados por meio de filtros dinâmicos, possibilitando a análise por ano, nível de senioridade, tipo de contrato e tamanho da empresa. Além disso, apresenta métricas importantes como salário médio, salário máximo, cargo mais frequente e total de registros.

A aplicação foi construída com:

Streamlit para criação da interface interativa e dos filtros dinâmicos

Pandas para carregamento, tratamento e análise dos dados

Plotly Express para visualização de dados com gráficos interativos

Este projeto reforça habilidades em:

Análise e manipulação de dados com Python

Criação de dashboards interativos

Visualização de dados

Desenvolvimento de aplicações com Streamlit

Estruturação de projetos para portfólio em Data Science
