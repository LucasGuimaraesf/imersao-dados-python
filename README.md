# Dashboard de Análise de Salários na Área de Dados

## Resumo
Dashboard interativo desenvolvido em Python para análise de salários na área de dados, utilizando Streamlit, Pandas e Plotly. A aplicação permite explorar dados salariais por meio de filtros dinâmicos e visualizações interativas, facilitando a análise de tendências e características do mercado de trabalho em dados.

---

## Sobre o projeto

Este projeto foi desenvolvido como parte da **Imersão de Python com Dados da Alura**, com o objetivo de aplicar conceitos de análise de dados e desenvolvimento de dashboards interativos utilizando Python.

O dashboard permite explorar informações salariais da área de dados por meio de filtros dinâmicos, possibilitando a análise por:

- Ano  
- Nível de senioridade  
- Tipo de contrato  
- Tamanho da empresa  

Além disso, apresenta métricas importantes como:

- Salário médio  
- Salário máximo  
- Total de registros  
- Cargo mais frequente  

Também inclui visualizações interativas que facilitam a análise dos dados e a identificação de padrões e tendências.

---

## Tecnologias utilizadas

- Python  
- Streamlit  
- Pandas  
- Plotly Express  

---

## Funcionalidades

- Filtros interativos na barra lateral  
- Exibição de métricas principais (KPIs)  
- Gráfico de barras com os cargos mais bem pagos  
- Histograma da distribuição salarial  
- Gráfico de pizza com tipos de trabalho (remoto, presencial, híbrido)  
- Mapa com salário médio de Cientistas de Dados por país  
- Visualização completa dos dados em tabela
